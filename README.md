# MangoGram (Socket Messenger)

Console messenger using Python sockets and threading.

## Features

- Private messaging
- Multiple clients
- Timestamps
- Thread-based server

## Commands

- Type messages and press Enter
- `/exit` ‚Äî leave the chat

## Technologies

- Python
- socket
- threading

## How to run

1. python server.py
2. python client.py
